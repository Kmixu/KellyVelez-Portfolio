Exquisite Specialty Treats Business Package

This featured project holds a special place in my journey as a designer. I am incredibly proud of the work I accomplished here, as it showcases a complete branding package that I developed from the ground up. The project includes a custom-designed logo, front and back envelope designs, and a matching business card.

Each element was thoughtfully created to ensure consistency, professionalism, and visual appeal across all brand materials. From concept to execution, I took care to reflect the identity and values of the brand through typography, color choices, and layout.

This project allowed me to strengthen my skills in brand identity design and demonstrated my ability to bring a cohesive visual experience to life.

How I've Grown

This course has been a big part of my growth as a designer and student. When I started, I had only a basic understanding of design and coding. But now, I feel much more confident. I've learned how to create responsive websites using HTML and CSS, starting from a blank page. I understand how to keep my files organized, write cleaner code, and make sure my website works for everyone, including those with accessibility needs.

One of the best tools I learned was Flexbox. It helped me design layouts that adjust to different screen sizes, which makes my websites look better and work better on phones and desktops. I also learned how to use external CSS files and write semantic HTML to build stronger and more structured pages. On top of the technical side, I’ve also improved my eye for design—things like color combinations, font choices, and visual hierarchy now make more sense to me.

What helped me grow the most was the final project. I built my own website from scratch, and it brought everything together. I included a custom slideshow, special bullet styles, and a fully styled layout. While creating the project, I ran into several problems—but every time I solved one, I learned something new. It really pushed me to think critically and figure out how things work on my own.

Overall, I've come a long way in this class. I not only gained technical skills but also grew in confidence. Now, I can build something real and functional—and I know how to fix it if something goes wrong. That's a big win for me.
